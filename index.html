    • Analysing India's Air Quality using Python

    PROJECT OBJECTIVE:

    • Conducting a study on highly polluted cities, and analyzing major pollutant components.
    • Performing time-series analysis with a ARIMA model, using computed orders, to forecast India's AQI in 2021.

    KEY QUESTIONS:
    • Analyze overall air quality trend over a specific time period and identify seasonal patterns.
    • Determine major pollutants contributing to poor air quality in a particular region and explore geographical patterns across different cities or regions within the country.
    Download data set here.
    ANALYSIS STEPS:
  • Data Preparation:
    • Conducted research using Python to address data profiling, integrity issues, quality and data transformation for a cohesive dataset.
    • Transformed data into a consistent and unified format.
  • Analysis:
    • Merged, analyzed, and added new columns with conditional logic, functions, loc.
    • Exploring relationships using Correlation, Scatterplot, Catplot etc. and Time Series Analysis using ARIMA Model.
  • Recommendations:
    • Strengthen regulations to control industrial and vehicular emissions.
    • Promote sustainable agricultural practices to reduce stubble burning and its impact on air quality.

    VISUALIZATIONS:
    Correlation Matrix BTX has the lowest correlation with AQI- which is perfectly in sync with the AQI calculation formula. The air quality index is composed of 8 pollutants ((PM10, PM2.5, NO2, SO2, CO, O3, and NH3), which does not directly account for BTX.

    Time Series Forecasting Using ARIMA Model The SARIMAX model with the specified parameters (ARIMA(5, 1, 3)) does not seem to be a good fit for the data based on the non-significant coefficients and the results of the diagnostic tests. You may need to consider adjusting the model parameters or exploring other models to achieve a better fit to the data.
